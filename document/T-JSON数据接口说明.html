超龙API接口文档

备注：采用restful设计风格，除账号模块外，其他模块的接口使用都需要使用token验证。
示例：http://localhost:8081/drivingAPI/user?tocken=qwes321asfqedsada43

1、账号模块

1.1、登录

方法

POST http://localhost:8081/drivingAPI/login

功能

用户通过账号进行登录。

url参数

无

表单参数

字段 类型 必选 说明 示例 最后更新时间
phone string 是 账号（手机号） "17197929204" 正常
password string 是 密码 "jovan123" 正常

返回结果

{
    "code":200,
    "status":"success",
    "message":"提交成功",
    "data":{
        "user_id":"asdasd-fasfasf-asfdasdas",
        "username":"聪聪",
        "image":"http://localhost:8081/image/heading.jpg",
        "district":"红谷滩新区",
        "tocken":"qwes321asfqedsada43"
    }
}

返回参数说明

字段 类型 说明 最后更新时间
code int 错误代码 正常
status string 状态 正常
message string 错误消息 正常
user_id string 用户id 正常
username string 用户昵称 正常
image string 用户头像 正常
district string 用户地区 正常
tocken string 用户有效凭证 正常

错误码说明

错误代码 错误英文 说明
200 Success 数据请求成功

更多错误代码请见错误代码对照表
